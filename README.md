# 🎮 Juego del Número Secreto

¡Bienvenidos, que se diviertan! 🎉

---

## 🎯 Objetivo del Juego
Su objetivo es **adivinar el número secreto** en la menor cantidad de intentos posibles. ¡Pon a prueba tu intuición y estrategia!

## 🔍 ¿Cómo jugar?
1. El sistema generará un número aleatorio de forma oculta.
2. Ingresa tus intentos en la interfaz o consola.
3. El juego te dará pistas indicando si el número secreto es mayor o menor que tu respuesta.
4. ¡Sigue intentando hasta dar con el número correcto!
